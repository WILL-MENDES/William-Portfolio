Minhas Habilidades

HTML

HTML é a linguagem de marcação usada para criar páginas web. É composta por uma série de tags (etiquetas) que descrevem o conteúdo e a estrutura da página, como cabeçalhos, parágrafos, imagens, links e formulários.

CSS

CSS é uma linguagem usada para estilizar e formatar páginas web escritas em HTML. O CSS permite controlar a aparência e o layout da página, como as cores, fontes, tamanhos, posicionamento e espaçamento dos elementos.

Javascript

é usada para criar interatividade e dinamismo em páginas web. Ela permite que adicionem comportamentos complexos e interativos às páginas, como animações, exibição de dados em tempo real, efeitos visuais e muito mais.

Photoshop

Photoshop é amplamente utilizado por designers, artistas gráficos, fotógrafos e profissionais de mídia para criar, editar e manipular imagens digitais em uma ampla variedade de formatos.

Wordpress

é um sistema de gerenciamento de conteúdo gratuito e de código aberto usado para criar e gerenciar sites. ele permite que os usuários criem facilmente conteúdo, como artigos, páginas, blogs por meio de uma interface intuitiva e amigável.

Responsivo

é uma técnica de design de sites e aplicativos que permite que eles se adaptem a diferentes tamanhos de tela, resoluções e dispositivos, proporcionando uma experiência de usuário otimizada em qualquer dispositivo.
